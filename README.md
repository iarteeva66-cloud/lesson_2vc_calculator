# Семейный бюджет

Небольшой статический сайт для учёта семейных доходов и расходов: баланс, список операций, диаграмма по категориям, фильтр по году и месяцу, тёмная тема, отмена шагов и восстановление после полной очистки. Данные хранятся в **localStorage** браузера (без сервера и регистрации).

## Структура репозитория

| Файл | Назначение |
|------|------------|
| [index.html](index.html) | Приветственный лендинг и переход в приложение |
| [app.html](app.html) | Рабочий интерфейс учёта (бывшая главная страница) |

## Как открыть

1. Склонируйте репозиторий (`git clone …`) или скачайте архив папки. Если Git не установлен, достаточно скопировать каталог проекта.
2. Откройте `index.html` в браузере двойным щелчком или через меню «Файл → Открыть».
3. Нажмите **«Открыть учёт»** — откроется `app.html` с полным функционалом.

Для разработки удобнее поднять локальный сервер (иначе в некоторых браузерах возможны ограничения для `file://`):

```bash
# при установленном Python 3
python -m http.server 8080
```

Затем в браузере: `http://localhost:8080/`

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и отправьте туда файлы (`git push`).
2. В настройках репозитория: **Settings → Pages → Branch** (например `main` / корень или `/docs`).
3. В `index.html` при желании добавьте ссылку на репозиторий в блоке под кнопкой «Открыть учёт».
